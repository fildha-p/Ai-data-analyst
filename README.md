# AI Data Analyst

AI Data Analyst is a Streamlit application that allows users to upload CSV files, explore datasets, visualize trends, and generate AI-powered insights in seconds.

---

## 🚀 Features

- Upload CSV datasets
- Preview data (rows, columns, missing values)
- Automatic column detection (numeric, categorical, date)
- Summary statistics for numeric and categorical data
- Data visualizations:
  - Bar charts
  - Line charts
  - Histograms
- AI-generated insights and recommendations
- Natural language Q&A on your dataset

---

## 🧠 How It Works

1. Upload a CSV file  
2. The app analyzes the dataset using pandas  
3. It generates summary statistics and visualizations  
4. A compact dataset summary is sent to the AI model  
5. The AI returns:
   - Business insights  
   - Key findings  
   - Recommendations  
6. Users can ask questions about the dataset in plain English
